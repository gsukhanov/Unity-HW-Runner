# Unity-HW-Runner
Project for uni, a runner game made with Unity

# Описание
Простенький раннер с тремя дорожками, где вы управляете золотым шариком и уворачиваетесь от чёрных параллелепипедов.

# Управление
A, D - сменить дорожку.
Пробел - прыжок.
Во время прыжка менять дорожку нельзя.

# Настройки
В верхней панели во вкладке Tools есть инструмент Settings Editor. В нём доступны следующие настройки:
1. Максимальное здоровье игрока
2. Высота прыжка
3. Гравитация
4. Начальная скорость
5. Максимальная скорость
6. Время для достижения максимальной скорости
7. Средний урон препятствия
8. Частота появления препятствий

# Требования (основные и дополнительные)
Реализовано движение по дорожкам.
Реализован прыжок на фиксированную высоту.
В игре используется система ввода `InputEvents`.
Скорость движения постепенно растёт.
В игре присутствует тень игрока - вспомогательный объект, находящийся рядом с игроком, но всегда по центру, чтобы от него отсчитывать расстояния.
Камера догоняет тень игрока, тем самым подчеркивая изменения скорости. Скорость камеры растет экспоненциально, а игрока и тени - линейно, до одного значения за равное время, после чего камера догоняет тень, пока не поравняется с ней.
В левом верхнем углу указано здоровье игрока.
Размер препятствий случайный.
Урон препятствий случайный.
Положение препятствий на одной из дорожек случайное.
Препятствия удаляются при удалении от тени игрока.
Уровень достраивается при приближении к его краю а старые компоненты удаляются.
При столкновении с препятствием оно разрушается, а игрок получает урон.
После падения здоровья ниже 0 игрок умирает и появляется кнопка, перезапускающая игру.
